# 隐私政策

**生效日期：** 2026年2月5日  
**组织名称：** Tingly Dev  
**产品名称：** Tingly Box  

欢迎使用 **Tingly Box**（以下简称“本产品”）。Tingly Dev（以下简称“我们”或“我方”）高度重视您的隐私权。本隐私政策旨在向您清晰说明我们在您使用 Tingly Box 过程中如何处理您的个人信息。

## 1. 我们不收集任何个人数据

Tingly Box 的设计原则是尊重用户隐私。**我们不会以任何形式收集、存储、传输或处理您的任何个人身份信息（Personal Data）**，包括但不限于：

- 姓名、电子邮件地址、电话号码  
- IP 地址、设备标识符（如 IDFA、Android ID）  
- 位置信息、浏览记录、使用行为数据  
- 任何可通过技术手段识别到您个人的信息  

本产品在本地设备上运行，所有数据（如有）均保留在您的设备中，不会上传至任何服务器或第三方服务。

## 2. 第三方服务

Tingly Box **不集成任何第三方分析工具、广告 SDK、云服务或追踪技术**（例如 Google Analytics、Firebase、Facebook SDK 等）。因此，没有任何第三方能够通过本产品获取您的信息。

## 3. Cookie 与追踪技术

由于 Tingly Box 是一款本地运行的应用程序（或离线工具，根据实际情况可调整描述），**我们不使用 Cookie、网络信标、像素标签或其他追踪技术**。

## 4. 儿童隐私

我们不会有意收集 13 岁以下儿童的任何信息。如您发现我们无意中收集了此类信息，请联系我们（见下文），我们将立即删除。

## 5. 隐私政策的更新

尽管我们目前不收集任何数据，但未来若产品功能发生重大变化（例如新增需要收集数据的功能），我们将：
- 提前更新本隐私政策；
- 在应用内或通过其他合理方式通知用户；
- **仅在获得您明确同意后**才开始收集任何个人信息。
